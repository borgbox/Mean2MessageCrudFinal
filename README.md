CRUD Mean stack 2 com webpack

<h2>Instalar</h2>
<br>
<p>npm install
<br>

<h2>Buildar</h2>
<br>
<h4>Desenvolvimento</h4>
<br>
<p>npm run build
<br>
<h4>Produção</h4>
<br>
<p>npm run build

<h2>Executar</h2>
<br>
<p>npm start

<br>
<p>heroku config:set VARIAVEL='Valor'

<h4>Windows</h4>
<br>
<p>set VARIAVEL='Valor'
